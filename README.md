# BingSearchApi_GoogleAppsScripts
Use Bing Search Api in Google Sheets using Apps Script

Requisites
Have at least a trial account of Microsoft Azure (1,000 queries per month for the Bing Search Api)
Have or create a Bing Search V7 Api resource and have access to your Subscription Key (Manage Keys > Key1) and Endpoint information
add img
Have a Google Account and a Google Sheets spreadsheet, you need to authorize the execution of the script with your Google Account

There is a JSON response included in the repository as an example of the response given by the API
